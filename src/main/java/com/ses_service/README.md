# Service Entry Sheet (SES) Service

A Spring Boot REST API for managing Service Entry Sheets.

## Overview
This service provides CRUD operations for Service Entry Sheets used in service procurement workflows.

## Features
- Create Service Entry Sheet
- Get Service Entry Sheet by ID
- Get all Service Entry Sheets
- Update Service Entry Sheet
- Delete Service Entry Sheet

## Architecture
- Controller: Handles REST APIs
- Service: Contains business logic
- Repository: JPA-based persistence layer

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- H2 / MySQL
